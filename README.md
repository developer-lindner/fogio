fogio
=====

Just a FogBugz to Planio [very specific] importer.

Basic usage
=====

1 - Install composer to manage dependencies: https://getcomposer.org/

2 - Install the dependencies

```shell
composer install
```

3 - Run the command via CLI

```shell
php fogio-cmd.php
```
